# 🎬 Movie Ticket Booking System

## Overview

A full-stack Movie Ticket Booking System built using React, Spring Boot, PostgreSQL, JWT Authentication, and Docker.

Users can browse movies, select theaters and show timings, book seats, and manage bookings. Admins can monitor bookings, view theater statistics, and manage cancellations.

---

## Tech Stack

### Frontend
- React.js
- JavaScript
- CSS

### Backend
- Spring Boot
- Java
- REST APIs

### Database
- PostgreSQL

### Security
- JWT Authentication
- Role-Based Access Control

### Deployment
- Docker
- Docker Compose

---

## Features

### User Features
- User Login
- Movie Selection
- Theater Selection
- Show Timings
- Seat Booking
- Cancel Booking
- User Dashboard

### Admin Features
- View All Bookings
- Theater Statistics
- Cancel Any Booking
- Admin Dashboard
